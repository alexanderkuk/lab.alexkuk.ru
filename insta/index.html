Поиск тематических аккаунтов в Инстаграме

Как найти самые влиятельные профили заданной тематики.
2016-10-01

Предположим, компания выпускает щётки для скалолазания и хочет запустить рекламу в Инстаграме. Для этого нужно найти популярных скалолазов, прислать им щётки и попросить написать отзывы.

Первым делом, соберём максимально полный список скалолазов в Инстаграме. При оформлении подписки выводится список из 100 похожих аккаунтов:

Вручную найдём 10 «посевных» аккаунтов скалолазов. Для каждого скачаем список похожих. Получим 230 аккаунтов. Повторим процедуру, скачаем для них списки похожих. Получим 2300 аккаунтов. Будем повторять процедуру пока не наберём 40 000 профилей. Получается роскошный граф, в котором вершина — это пользователь, а ребро появляется, если аккаунты похожи.

Вершины разных цветов соответствуют разным группам пользователей: скалолазы, фотографы, альпинисты, лыжники. Отберём нужных пользователей и отсортируем их по числу подписчиков.

В списке много организаций: производители снаряжения, одежды, национальный парки, спортивные журналы. Часто, когда на аватарке есть лицо, пользователь — человек, иначе компания. Для поиска лиц используем несложный код на OpenCV. Получается список из нескольких сотен аккаунтов.
